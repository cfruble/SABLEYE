# SABLEYE
Simulation and Analysis of Burnup and Lifecycle Elemental YiElds

Implementation for NERS570 Project by Colin Ruble and Qasim Siddiq

To get started with this package, run the following commands:
```
$ git clone https://github.com/cfruble/SABLEYE.git
$ bash setupData.sh
```
This will download the ENDF files and process then into a format the the program can use.  The total size for all of the data when unzipped is about 3.0 GB.
